# Data Dictionary — carbon_qml_project

## Overview

This folder contains the input dataset and derived outputs for the replication archive.
All variables are described below with their source, transformation, and role in the analysis.

---

## Input Data

### `master_dataset.csv`

**Description:** Daily panel of EU ETS price and energy market variables.
**Rows:** 1,302 trading days  
**Period:** 2019-01-03 to 2023-12-29  
**Frequency:** Daily (trading days only; weekends/holidays excluded)

| Column | Full Name | Unit | Transform | Source | Role |
|--------|-----------|------|-----------|--------|------|
| `date` | Trading date | YYYY-MM-DD | None | ICE/Eurostat | Index |
| `EUA_return` | EU Allowance log-return | Dimensionless | ln(Pₜ/Pₜ₋₁) | ICE Futures/Sandbag | **Target variable** |
| `GAS_return` | TTF Natural Gas log-return | Dimensionless | ln(Pₜ/Pₜ₋₁) | Investing.com | Primary energy driver |
| `OIL_return` | Brent Crude Oil log-return | Dimensionless | ln(Pₜ/Pₜ₋₁) | FRED DCOILBRENTEU | Macro energy proxy |
| `COAL_return` | ARA Coal log-return | Dimensionless | ln(Pₜ/Pₜ₋₁) | Platts/ICE | Fuel-switching signal |
| `ELEC_return` | German/European electricity log-return | Dimensionless | ln(Pₜ/Pₜ₋₁) | EEX/ENTSO-E | Power sector proxy |
| `IP_return` | Industrial Production YoY growth | % | Step-ramp interpolation (monthly→daily) | Eurostat STS_INPR_M | Activity proxy |
| `CPI_return` | HICP Inflation YoY growth | % | Step-ramp interpolation (monthly→daily) | Eurostat PRC_HICP | Macro cost-push |
| `POLICY_dummy` | ETS regulatory event indicator | Binary (0/1) | 7-day symmetric window around event date | EUR-Lex + ECB | Regulatory shock |
| `PHASE_dummy` | EU ETS Phase indicator | Binary (0/1) | Step: 0=Phase 3 (2019–2020), 1=Phase 4 (2021–2023) | EU Commission | Structural regime |

**Notes:**
- All price series use log-returns to achieve stationarity (confirmed by ADF and KPSS tests; Table 3 in manuscript).
- Monthly variables (IP, CPI) use step-ramp interpolation with strict no-look-ahead enforcement at each fold boundary.
- POLICY_dummy = 1 for dates within ±7 days of EU ETS regulatory events (Phase 4 reform, MSR review announcements).
- Original raw price series (EUA spot, TTF front-month, Brent, ARA Coal, German electricity day-ahead) are **not included** due to data provider licensing restrictions. Log-returns are derived quantities suitable for replication.

## Output Data (generated by pipeline)

### `metrics_fold_level.csv`
Generated by: `04_statistical_validation.py`  
**Rows:** 999 (9 models × 37 folds × 3 horizons)

| Column | Description |
|--------|-------------|
| `model` | Model identifier (`qk_svr`, `bilstm`, `gru`, `transformer`, `emd_lstm`, `rbf_svm`, `laplacian_svm`, `xgboost`, `lightgbm`) |
| `fold_id` | Sequential fold identifier (1–111) |
| `horizon` | Forecast horizon in trading days (1, 5, 22) |
| `test_start` | First date of the test window (YYYY-MM-DD) |
| `rmse` | Root mean squared error of log-returns (fold-level) |
| `mae` | Mean absolute error of log-returns |
| `mape` | Mean absolute percentage error (NaN when returns ≈ 0) |
| `da` | Directional accuracy (%) — 0 or 100 for single-step folds |
| `sharpe` | Sharpe ratio of long-short strategy (annualised via √252) |
| `n_obs` | Number of test observations in fold |

### `crisis_subperiod.csv`
Generated by: `04_statistical_validation.py`  
**Rows:** 108 (9 models × 3 horizons × 4 regimes)

| Column | Description |
|--------|-------------|
| `regime` | `pre_crisis`, `crisis_onset`, `peak_crisis`, `post_crisis` |
| `rmse_mean` | Mean RMSE across regime folds |
| `rmse_std` | Standard deviation of RMSE across regime folds |

**Regime boundaries** (structural breaks from Bai-Perron + Zivot-Andrews tests):
- Pre-Crisis: 2019-01-03 – 2021-01-31 (training window)
- Crisis-Onset: 2021-02-01 – 2022-01-31
- Peak Crisis: 2022-02-01 – 2022-12-31 (n=12 folds for H=5)
- Post-Crisis: 2023-01-01 – 2023-12-29
